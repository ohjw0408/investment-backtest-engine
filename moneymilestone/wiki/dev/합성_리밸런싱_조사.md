# 합성 데이터 · 리밸런싱 현상 조사 (2026-06-06)

> BUG-SYNTH-CORR(합성 상관 복원) 배포 후, 오너가 "합성 장기 시뮬서 리밸런싱하면
> 자산이 늘고 MDD는 안 줄어든다(밴드 넓힐수록 더 늘어남)"는 이상현상 제기 → 심층 조사.
> **결론: 리밸런싱이 수익을 더하는 것은 (분산자산에선) 실제 금융현상이며 버그 아님.**
> 코드 수정 보류(실데이터가 진짜이므로 손대면 신뢰도 훼손) — 오너 결정.

## 1. 오너가 본 현상

합성 가상데이터(use_synthetic)로 GLD+SCHD+SPY / GLD+SPY 40년 등비중:
- 무리밸 < 매월 < 밴드1% < 밴드30% (리밸할수록·밴드 넓을수록 최종자산↑, 예 37억→44억)
- MDD는 모드 무관 거의 불변
- "리밸하면 보통 수익 줄고 MDD 줄어야 하는데 반대다 → 버그"

## 2. 조사 (전부 서버 178.105.84.213 실DB·읽기전용, 일부 순수 numpy)

### 2-1. 집행 단계 가치보존 (코드)
`Portfolio.buy/sell` + `total_value`가 같은 price_dict 사용 → 리밸 시 포트가치 **수학적으로 정확히 불변**. 정수절삭은 가치 감소(누수 아님). **트레이드가 가치를 만들지 않음.** → 종료자산 증가는 누수 아니라 **allocation(리밸런싱 프리미엄) 효과.**

### 2-2. 상관 복원은 end-to-end 정상 (BUG-SYNTH-CORR 수정 검증)
서버 3자산 다중세그먼트 합성 실현상관: **SCHD-SPY 0.03→0.922**, GLD-SPY 0.23, GLD-SCHD 0.21. 내 조건부 다변량 수정 확실히 작동.

### 2-3. ⚠️ FX(환율) 오염 — 상관이 KRW 공간서 양수로 부풀려짐
`get_price` 기본 `apply_fx=True`(KRW) → 모든 미국자산이 USD/KRW 공통인자 공유 → `Cov` 에 `Var(r_fx)` 양수항 가산 → 상관 인위적 상승.

| 쌍 | KRW(apply_fx=T) | USD(apply_fx=F) |
|---|---|---|
| **TLT-SPY** | **+0.231** | **−0.156** (음수=정상) |
| GLD-SPY | +0.248 | +0.060 |
| SCHD-SPY | +0.914 | +0.894 |

웹 확인: TLT-SPY 역사적 음의 상관(2013 −0.80, 전기간 ≈−0.25, 최근 +전환). **저상관·음상관 자산(채권·금)일수록 FX가 부호를 뒤집음.** SCHD-SPY는 이미 고상관이라 FX 영향 미미(그래서 BUG-SYNTH-CORR 수정이 멀쩡해 보였음). → 헤지·MDD감소가 안 보이는 이유 = KRW 양상관이라 헤지 메커니즘이 죽음.

### 2-4. ⭐ 리밸 수익증가는 실데이터서도 발생 = 실제 금융현상
순수 GLD+SPY **실데이터** 2004~2026(합성 전혀 없음), 깨끗한 numpy 리밸:

| | always(잦은리밸) | band30(넓은밴드) |
|---|---|---|
| **실제 순서** | +17.0% | **+57.6%** |
| **셔플(시간순서만 파괴=i.i.d.화, 상관·분포 보존)** | +17.0% | **+0.0%** |

- **always +17%는 실제=셔플 동일** → 순수 변동성 수확(시간순서 무관, 실재). σ²·분산효과 기반.
- **band30 +57.6%는 실제 시간구조(금/주식 다년 로테이션 사이클)에서 100% 나옴** → 셔플하면 0. 금 2004~11 불장→주식 2011~21 불장, 넓은밴드가 회전을 잡아 고점 매도·저점 매수.
- **합성(i.i.d.)은 band30 중앙값 0 + 고분산**(12윈도우 −19%~+28%) → 실제 사이클을 못 살려 **노이즈**. 즉 합성은 band를 **과대가 아니라 과소·불안정**하게 냄.

### 2-5. 실제 엔진(분석기 p50) 재현 — GLD+SPY 40년 합성, 초기 1천만·월적립0·세금OFF

| 모드 | step=6(n=41) | step=1(n=41) | MDD |
|---|---|---|---|
| none | +0% | +0% | −39.7% |
| monthly | +15.7% | +17.4% | −37.8% |
| band1% | **+21.1%** | +21.6% | −37.7% |
| band5% | +17.0% | +23.2% | −37.7% |
| band30% | +8.8% | +5.2% | −38.6% |

- **리밸 +15~23% 증가 재현**, MDD ~2%p만 변화(거의 평탄) — 오너 현상 robust 부분 재현.
- 단 **band1%(잦음)이 최고·band30%(드뭄) 최저** → 오너가 본 "band30 최고/단조증가"는 **재현 안 됨**(band30은 고분산이라 런마다 순위 바뀜). band30 순위는 노이즈.

## 3. 결론

1. **리밸런싱이 수익을 더하는 것 = 실제 금융현상**(분산자산). always=변동성수확(실재), band=실제 다년 사이클 회전(실재, 합성 실데이터 모두 +). 오너 직관 "리밸하면 수익 줄어야"는 **같은클래스/추세 자산엔 맞지만 분산자산엔 안 맞음.** → **버그 아님. 코드 수정 안 함**(실데이터가 진짜라 손대면 신뢰도↓ — 오너 결정).
2. **MDD가 안 줄어드는 핵심 원인 = FX 양상관**(2-3). USD 언더라잉이면 음상관 살아나 헤지·MDD감소 보임. **모델링 결정 사항**(미결).
3. **합성 band 리밸은 i.i.d.라 고분산 노이즈**(실제 다년 사이클 미반영) → band 결과 저신뢰. 블록 부트스트랩이면 사이클 보존돼 안정화되나 **프리미엄을 없애진 않음**(프리미엄 실재). 현재는 미구현.

## 4. 미결 / 향후 (전부 보류 — 오너 판단 대기)

- **(결정) FX 상관 공간:** KRW(현행, 환헤지 안 한 한국투자자 실경험에 충실하나 헤지 가려짐) vs USD 언더라잉+FX 별도인자(교과서 분산효과·음상관 표현). 미결.
- **(개선) 블록 부트스트랩:** 합성을 i.i.d.→실제 사이클 보존으로. band 노이즈 감소·현실화. 단 리밸 프리미엄은 실재라 유지. 작업 큼.
- **(경고) 합성구간 band 리밸 저신뢰** 사용자 안내 고려.
- BUG-SYNTH-CORR 수정(조건부 다변량, KRW 공간 추정)은 **배포·유지**. 같은클래스 상관은 정확(SCHD-SPY 0.92). 분산자산은 FX-KRW 상관을 충실 재현(헤지 가려짐은 FX 본질).
