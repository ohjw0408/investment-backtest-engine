# Money Milestone — 앱 개요

## 한 줄 정의

자산배분 투자자가 감정 없이, 원칙대로, 오래 투자할 수 있게 돕는 **포트폴리오 시뮬레이터 + 개인재무관리 앱**.

## 핵심 가치 제안

> "얼마나 벌었는지"가 아니라 "얼마나 오래, 안전하게 굴릴 수 있는지"를 계산해준다.

1. **리밸런싱 부담 제거** — 어제 사야 할 종목, 양, 금액을 알림으로 알려줌
2. **FOMO 극복 도구** — 내 포트폴리오가 역사적으로 얼마나 버텼는지 데이터로 확인
3. **한국 세금 완벽 반영** — 연금저축/IRP/ISA × 양도소득세/배당소득세 × 종합과세 모두 계산
4. **국내 유일 통합 시뮬레이터** — 백테스트 + 은퇴 인출 + 배당 역산 + 세후 수익 한 곳에서

## 해결하는 문제

### 1차 타겟: 기존 자산배분 투자자의 실무 고통
- 리밸런싱 계산이 번거롭고 지속하기 어려움
- 복잡한 절세 계좌(연금저축, IRP, ISA, 위탁) 관리가 너무 어려움
- 폭등장에서 FOMO → 충동매매 → 전략 이탈

### 2차 타겟: 예·적금 사용자의 자산배분 시장 유입
- 자산배분 수익률이 낮다는 오해 (실제: 연 7~20%)
- "주식 = 위험" 심리적 장벽
- 이를 데이터로 설득 → 시장 파이 자체를 키움

## 타겟 사용자

1. **40-50대 직장인** — 은퇴 준비, 체계적 노후 설계 필요
2. **노후가 불안한 주부** — 가계 자산 안정적으로 운용하고 싶음
3. **군 전역자 / 청년층** — 목돈 생긴 후 첫 재테크 시작
4. **예·적금 고객** — 저금리에 지쳤지만 주식은 무서운 사람
