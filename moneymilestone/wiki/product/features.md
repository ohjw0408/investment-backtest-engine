# 기능 목록

> ✅ 2026-05-30: 배당 데이터 0 버그는 `ETF_BACKFILL § Phase 6.0 Stage A` 서버 적용으로 해소. 배당 의존 기능(배당 역산, 종합과세)은 정상 배당 데이터 기준 재검증 필요. → [[dev/status]]

## 핵심 계산기 (Core Calculators)

| 기능 | 상태 | 설명 |
|---|---|---|
| 포트폴리오 백테스팅 | ✅ 완료 | n년간 포트폴리오 성과 분석 (CAGR, MDD, 샤프) |
| 포트폴리오 시뮬레이션 (롤링) | ✅ 완료 | 역사 속 모든 시작점에서 n년 투자 결과 다발 시각화 |
| 목표 배당금 역산 계산기 | ✅ 완료 | 월 X원 배당 수령 목표 → 필요 초기자본/월납입 역산 |
| 세금 계산기 | ✅ 완료 | 연금저축/IRP/ISA/위탁 × 양도세/배당세/종합과세 |
| 은퇴 적립 시뮬레이션 | ✅ 완료 | n년 적립 후 자산 예측 |
| 은퇴 인출 시뮬레이션 | ✅ 완료 (Tax Phase 2d) | 생존 확률, 최대 안전 인출액 계산 |

## 고급 기능 (Advanced)

| 기능 | 상태 | 설명 |
|---|---|---|
| 리밸런싱 알림 | 💡 미구현 | 보유 종목/비중 입력 → 비중 이탈 시 매수/매도 수량 알림 |
| TDF 시뮬레이션 | 💡 미구현 | 글라이드패스 커스텀, 나이에 따른 자동 비중 조정 |
| 절세매도 (Tax Harvest) | ✅ 완료 | 연말 손익통산으로 세금 최소화 |
| 금융소득 종합과세 경고 | ⚠️ 부분 (Phase 2e) | 엔진 완료, 백테스트만 배선. 자동산출/전탭배선 미완 + Stage A 정상 배당 데이터 기준 재검증 필요 |

**중단기:**
- D4 거래수수료 설정 (1~2일)
- D5 인플레이션 검증 + 실질 생활비 (2~3일)
- A4 종목 상세 개선 + 시간봉 차트 (3~4일)
- B1 포트폴리오 즐겨찾기 (2~3일)
- B2 자산 추이 + 홈 토글 (1~2일)
- B3 리밸런싱 경고 밴드 (1일, B2 선행)

- C2 자산군별 수익률 비교 (2~3일)

- D2 연금 통합 계산기
