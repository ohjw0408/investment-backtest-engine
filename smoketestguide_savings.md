# 절세액 표시 P1 — 브라우저 스모크 테스트 가이드

투자계산기 **절세액 표시**(위탁가정세금·실제세금·절세액 + 절세매도 추가절감)를
실제 브라우저에서 손으로 확인하는 절차.

> 작성: 2026-06-02 (Claude). 엔진·서버 API는 결정론 테스트(L-SAVE 24종)+실데이터
> end-to-end로 검증 완료. 이 문서는 **결과 패널이 화면에 실제로 그려지는지** 육안 확인 보완 절차다.
> 대상 커밋: `03f28cb` 이후. 서버: https://moneymilestone.co.kr/calculator

---

## 절세액 패널이 어디 뜨나 (먼저 숙지)

실행 결과 화면에서 **"계좌별 종료 자산"** 카드 안, **연두색 박스**:

```
💰 세금 절감 효과 (중앙값 기준)
┌─────────────────┬──────────────┬──────────────┐
│ 전체 위탁 가정 세금 │   실제 세금    │    절세액     │
│   ₩2,623,025    │  ₩1,953,525  │  약 ₩669,500 │
└─────────────────┴──────────────┴──────────────┘
  ISA      위탁가정 ₩1,320,200 · 실제 ₩650,700 · 절세 ₩669,500
  위탁     위탁가정 ₩1,302,825 · 실제 ₩1,302,825 · 절세 ₩0
  ※ 근사치 — 금융소득종합과세 가산·연금 인출세 미반영…
```

절세매도(GH) 켠 위탁계좌가 있으면 그 아래 **주황색 박스** 추가:
```
📉 절세매도(연 250만 공제) 추가 절감          약 ₩550,000
```

## 사전 조건 (공통 — 모든 테스트 전 1회)

1. **세금 설정**: 세금설정 화면에서 — 나이 **40** · 연간 근로소득 **50,000,000** · ISA 유형 **일반형**.
2. **세금 ON**: 투자계산기 세금 적용 토글 켜기. (절세액 패널은 세금 ON일 때만 뜸)
3. **기간(투자 연수): 12년** (별 지정 없으면).
4. **배당: 재투자** / 리밸런싱: 없음(none).

## 공통 주의

- **ISA·연금·IRP는 국내상장 ETF만**(미국 직접상장 불가):
  - `458730` TIGER 미국배당다우존스 (= 국내상장 해외ETF, 배당 15.4%·양도 15.4%)
  - `360750` TIGER 미국S&P500 (동일 분류)
- **위탁은 미국 직접상장도 가능** — 절세매도(GH)는 **위탁 + 미국 직접상장 종목**에서만 의미.
  - 예: `SCHD`, `SPY` (티커 검색에서 미국 직접상장으로 잡히는 것)
- 화면 안 바뀌면 **강력 새로고침 `Ctrl+Shift+R`** (JS 캐시 `?v=20260602save` 확인).
- 실데이터 롤링이라 **정확한 ₩ 금액은 매번 다름.** PASS 판정은 **구조**(패널이 뜨나 / 숫자 부호 / 0 여부)로 한다.
- 절세액 = "세제혜택계좌(ISA/연금) **껍데기**가 위탁 대비 아낀 세금". 위탁 계좌는 항상 **절세 0**(정상).

## 테스트 S1 — ISA vs 위탁 절세액 (기본) ⭐

**목적:** 절세액 3종 패널이 뜨고, ISA는 절세 > 0, 위탁은 절세 0인지.

| 계좌 | 유형 | 초기자본 | 월적립 | 종목 | 절세매도 |
|---|---|---|---|---|---|
| 1 | ISA | 10,000,000 | 0 | 458730 100% | — |
| 2 | 위탁 | 10,000,000 | 0 | 458730 100% | OFF |

- **투자기간: 12년**, 세금 ON, 풍차 OFF.
- **기대:** 연두색 "💰 세금 절감 효과" 패널 표시.
  - 3종 숫자(전체 위탁가정세금 / 실제세금 / 절세액) 모두 표시.
  - 계좌줄: **ISA 절세 > 0** (₩ 수십만~수백만, 윈도우 따라 변동).
  - 계좌줄: **위탁 절세 = ₩0** (위탁가정 == 실제, 두 숫자 동일).
  - 합산 절세액 = ISA 절세 + 위탁(0).
- **PASS:** 패널 뜸 + ISA 절세 양수 + 위탁 절세 정확히 0 + 위탁의 위탁가정==실제.

## 테스트 S2 — 연금/IRP 절세액

**목적:** 연금/IRP 계좌도 절세액 표시(적립기 위탁가정 − 청산세 5.5%).

| 계좌 | 유형 | 초기자본 | 월적립 | 종목 | 절세매도 |
|---|---|---|---|---|---|
| 1 | 연금저축 | 10,000,000 | 0 | 360750 100% | — |
| 2 | 위탁 | 0 | 0 | 458730 100% | OFF |

- **투자기간: 12년**, 세금 ON.
- **기대:** 계좌줄에 **연금저축 절세 > 0** (위탁가정세금 − 실제 연금청산세 5.5%).
  - 연금 "실제" 숫자는 0이 아님(5.5% 청산세 — 오너 결정으로 현행 유지).
- **PASS:** 연금저축 줄 절세 양수, 실제세금 ≠ 0.
- **참고:** 투자계산기 연금/IRP는 **인출세 5.5%만** 반영. 1,500만 한도·나이별 세율은 **은퇴 탭(P3)**에서.

## 테스트 S3 — 절세매도(GH) 추가절감 주황 박스 ⭐

**목적:** 위탁 + 미국직접 종목 + 절세매도 ON이면 주황 "추가 절감" 박스가 뜨는지.

| 계좌 | 유형 | 초기자본 | 월적립 | 종목 | 절세매도 |
|---|---|---|---|---|---|
| 1 | 위탁 | 50,000,000 | 0 | SCHD(또는 SPY) 100% | **✅ ON** |

- **투자기간: 12년**, 세금 ON. (단일 위탁계좌도 OK — 멀티 아니어도 절세 패널 뜸)
- 종목은 **미국 직접상장**(알파벳 티커)이어야 GH 의미 있음. 국내ETF(458730)면 GH 0.
- **기대:**
  - 연두 패널: 위탁가정세금 > 0, **절세액 ₩0**(위탁이라 껍데기 절세 없음 — 정상).
  - 그 아래 **주황색 박스** "📉 절세매도(연 250만 공제) 추가 절감 약 ₩…" **양수** 표시.
- **PASS:** 주황 박스 뜸 + 추가절감 양수.
- **대조:** 절세매도 **OFF**로 같은 구성 재실행 → 주황 박스 **사라짐**.

## 테스트 S4 — 금(KRX금) 절세 0

**목적:** KRX 금현물은 위탁가정세금 0(비과세) → 절세 0 / 패널 미표시 확인.

| 계좌 | 유형 | 초기자본 | 월적립 | 종목 | 절세매도 |
|---|---|---|---|---|---|
| 1 | 위탁 | 10,000,000 | 0 | KRX금현물(KRX_GOLD) 100% | OFF |

- **투자기간: 12년**, 세금 ON.
- **기대:** 위탁가정세금 0 → 연두 패널 **안 뜸**(`brokerage_assumed_tax > 0` 조건 미충족) 또는 모든 절세 0.
- **PASS:** 절세 패널이 안 뜨거나, 떠도 위탁가정세금·절세 전부 0.

## 테스트 S5 — 세금 OFF 대조군

**목적:** 세금 OFF면 절세액 패널이 아예 없는지.

- 계좌 아무 구성 + **세금 OFF** + 투자기간 12년.
- **기대:** "💰 세금 절감 효과" 패널 **미표시**. 계좌별 종료자산만.
- **PASS:** 절세 패널 없음.

## 우선 확인 순서 (핵심)

1. **S1** — 절세 패널 뜨나 + ISA 절세 양수 + 위탁 절세 0 (가장 기본).
2. **S3** — 절세매도 주황 박스 뜨나 (위탁+US+GH ON).
3. **S2** — 연금/IRP 절세 표시되나.
4. S4·S5는 부가(금 0 / 세금OFF 미표시).

## 알려진 한계 / 주의

- **정확한 금액 검증 아님** — 엔진 정확성은 결정론(L-SAVE 24종)+서버 end-to-end로 검증됨.
  이 가이드는 **UI 렌더 육안 확인**.
- 절세액은 **근사치**: 금융소득종합과세 가산 생략, ISA 세후 재투자 가정, 연금 인출세는 은퇴탭만.
  → 패널 하단 회색 주석에 이 한계가 명시돼야 함(그 문구도 확인).
- 위탁 계좌 절세가 0이 아니면 **버그** (보고). 실데이터에서도 위탁가정==실제여야 함.
- 콘솔 에러("not valid JSON" 등) 뜨면 메시지 그대로 기록해 보고.
- 캐시 이슈 시 `Ctrl+Shift+R`.
