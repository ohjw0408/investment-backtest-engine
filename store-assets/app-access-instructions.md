# App Access Instructions Draft

Use this only if Google Play asks how reviewers can access the app.

## Access Type

Some features are available without login. Portfolio storage, alerts, and account settings require Google login.

## Reviewer Instructions

1. Open the app.
2. Browse public tools from Home or Tools.
3. To test account features, tap Google login.
4. After login, open My Assets, add a sample holding, and check Settings.

## Test Account

No shared password test account is used because login is handled by Google OAuth.

If Google requires a dedicated reviewer account, create a temporary Google account and enter it in Play Console App access. Do not commit the account credentials to this repo.

## Notes

Money Milestone is an informational portfolio and simulation tool. It does not provide investment advice, brokerage, trading execution, custody, lending, or crypto exchange services.
