Money Milestone 계정 및 데이터 삭제 안내

최종 개정일: 2026-07-04

Money Milestone은 사용자가 서비스 계정과 관련 데이터를 직접 삭제할 수 있는 방법을 제공합니다. 앱을 다시 설치하지 않아도 웹 브라우저에서 같은 절차를 진행할 수 있습니다.

1. 앱에서 계정 삭제하기

  1. Money Milestone 앱을 열고 Google 계정으로 로그인합니다.
  2. 하단 메뉴의 설정으로 이동합니다.
  3. 계정 탭에서 회원 탈퇴를 선택합니다.
  4. 확인 버튼을 누르면 계정과 관련 데이터가 삭제됩니다.

2. 웹에서 계정 삭제하기

  1. 웹 브라우저에서 https://moneymilestone.co.kr/settings#account로 이동합니다.
  2. 삭제하려는 계정과 같은 Google 계정으로 로그인합니다.
  3. 계정 탭에서 회원 탈퇴를 선택합니다.
  4. 확인 버튼을 누르면 계정과 관련 데이터가 삭제됩니다.

3. 삭제되는 데이터

  • Google 로그인 기반 계정 정보: Google 사용자 ID, 이메일, 이름, 프로필 사진 URL
  • 사용자가 입력한 자산 정보: 보유 종목, 수량, 평균단가, 계좌 유형, 자산 그룹
  • 저장한 포트폴리오, 세금 설정, 캘린더 설정, 관심 종목 및 알림 설정
  • 푸시 알림 토큰, 알림 규칙, 알림 발송 이력 등 계정에 연결된 서비스 데이터

4. 보관될 수 있는 데이터와 기간

계정 삭제 후 일반 서비스 데이터베이스에서는 관련 데이터가 삭제됩니다. 다만 보안, 장애 복구, 법적 의무 이행을 위해 필요한 서버 로그와 백업에는 일부 기록이 제한된 기간 남을 수 있습니다. 서버 백업은 최대 35일, 별도 보관 백업은 최대 90일 이내 순차적으로 삭제되며, 백업 데이터는 일반 서비스 제공이나 마케팅 목적으로 사용하지 않습니다.

5. 문의

로그인 문제 등으로 직접 삭제 절차를 진행하기 어려운 경우 moneymilestone@naver.com으로 문의해 주세요.
