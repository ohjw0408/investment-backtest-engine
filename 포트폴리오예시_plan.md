## 페이지 `/examples` "포트폴리오 예시" (분석 탭)
- mm_hero('포트폴리오 예시', '검증된 자산배분 전략을 골라 바로 분석·비교하세요. 비중·종목은 자유롭게 바꿀 수 있어요.').
- **지역 탭**: [🇺🇸 미국] [🇰🇷 한국] [⭐ 투자대가].
- 미국/한국 탭 = 성격 섹션(방어/헷지/배당/공격) 카드 그리드.
  - **전략 카드**: 이름 · 구성 미니바(비중) · 한줄설명 · 위험도 칩(방어↔공격, guru stance UI 재사용) · **공통 액션 3종**(아래).
- 투자대가 탭 = 기존 대가 카드 그리드(gurus 콘텐츠 편입) or "/gurus 전체보기" 진입.

### 공통 카드 액션 3종 (예시·대가 카드 전부 통일) ★오너 핵심
모든 포트폴리오 카드(자산배분 예시 + 투자대가)에 동일 3버튼:
1. **분석하기** → 백테(포트폴리오 분석) 핸드오프(`mm_bt_preload`+`/backtest` 자동실행). guru 기존.
2. **비교에 추가** → 포트폴리오 비교(/risk-return) 프리로드. **예시·대가 모두 신규**(대가엔 현재 없음 → 편입 시 추가).
3. **즐겨찾기 저장** → 내 포트폴리오. 대가=`mmSaveGuru` 기존 재사용, 예시=동일 패턴(`/api/portfolio/save`).
→ 핸드오프 헬퍼(`mm_bt_preload`·`mmSaveGuru`·신규 비교 프리로드)를 **공용화**해 예시·대가 공통 사용.
